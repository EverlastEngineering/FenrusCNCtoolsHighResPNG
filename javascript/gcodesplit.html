This website processes gcode files and splits them into smaller files, one file per tool, for folks without a BitSetter.

The BitSetter has been the best addon I got for my Shapeoko, and I highly recommend purchasing one. However, they aren't always in stock and shipping takes time... so for that case, this web application allows you to use Carbide Create's "Advanced V-Carve" feature even before the FedEx person shows up.

All computation happens local and inside your browser, no information is uploaded or sent over the internet. For large files this can take a few seconds before the results show up on the left.

Javascript diverges hugely between browsers; the Chrome browser is tested the most. I'll try to make others work on a best effort basis.

Found a bug? Have a suggestion? File them on github issues for this project
