Modulate gcode-over-gcode tool

This website lets you wrap existing gcode over another gcode file, to gcode for use with Shapeoko CNC machines

This currently only works if both gcode files are created with Carbide Create (regular or PRO)

All computation happens local and inside your browser, no information is uploaded or sent over the internet. For large files this can take 10+ seconds before the results show up.
