NCTILE - Tile GCode files

This website lets tile an existing gcode file into smaller areas for use with Shapeoko CNC machines

This tool requires that the zero of the original gcode is set in the lower left corner.

All computation happens local and inside your browser, no information is uploaded or sent over the internet. For large files this can take 10+ seconds before the results show up.
